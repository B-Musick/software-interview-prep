What is an API?

An API, also known as an Application Program Interface, is an exposed method (or collection of methods) for use by an external service to make use of your service. APIs can take many forms with varying levels of robustness and convenience to developers. When APIs are public, they will typically be accompanied with documentation and a promise to not make breaking changes which impact dependent services.
